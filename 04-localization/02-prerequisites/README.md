## Prerequisites
To be ready for this course, you should already be familiar with the following:

- Linear algebra
- How to take the derivative of polynomial functions
- How to calculate the mean, median, and standard deviation of a dataset
- Basics of working with lidar data
- C++ beginner to intermediate skills, such as working with header files, writing functions, and data structures & algorithms

If you're missing any of the above skills or want a refresher, you might consider checking out some of the following resources:
- [Multiplying Matrices](https://www.khanacademy.org/math/precalculus/x9e81a4f98389efdf:matrices/x9e81a4f98389efdf:multiplying-matrices-by-matrices/a/multiplying-matrices) (Khan Academy)
- [Differentiating Polynomials](https://www.khanacademy.org/math/old-ap-calculus-ab/ab-derivative-rules/ab-poly-diff/a/differentiating-polynomials-review) (Khan Academy)
- [Intro to Descriptive Statistics](https://www.udacity.com/course/intro-to-descriptive-statistics--ud827) (Udacity)
- The Sensor Fusion course of the Self-Driving Car Engineer Nanodegree program
- [C++ for Programmers](https://www.udacity.com/course/c-for-programmers--ud210) (Udacity)

There is also the [C++ Nanodegree program](https://www.udacity.com/course/c-plus-plus-nanodegree--nd213) from Udacity, although that is more than you will need here. A short upcoming mini-lesson will give you a basic check on if you have the C++ basics down that will be used throughout the course.
