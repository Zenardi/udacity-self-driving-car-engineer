## Additional Content

### stdout

Let's start with an easy one. In this quiz, I'm giving you a `main` function. All I want you to do is write `no more steering wheels` to stdout.
>Note: The workspace contains **challenges** and **solutions** directories, which contain all the challenges (1 through 5), and their solutions, respectively. For working on a **Challenge 1** you should navigate to the **challenge1** sub-directory within the **challenges** directory, and finish the task. Once you have finished writing your code, you can run it from the workspace terminal as follows:

```bash
cd /home/workspace/challenges/challenge1/
g++ main.cpp
./a.out
```

Let's understand what the above commands are doing:
* `cd /home/workspace/challenges/challenge1/` - Navigating to the challenge1 directory from the workspace terminal
* `g++ main.cpp` - Compiling the **main.cpp** file. 
	* If you have multiple **.cpp** files, then you need to compile all of them. For example, in **Challenge 2** you would have **main.cpp** and **printString.cpp**. To compile both of these files, you can run the command `g++ main.cpp printString.cpp`.
* `./a.out` - Running the executable file.
	* On succesful compilation, the compiler generates an executable file. In C++, this file is named **a.out** by default.
