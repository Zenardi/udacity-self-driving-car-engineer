## Summary

**Aligning Point Clouds with ICP and NDT**

This README file provides an overview of the key concepts learned in a lesson on aligning point clouds using Iterative Closest Point (ICP) and Normal Distributions Transform (NDT) algorithms, as well as working with point clouds in the CARLA simulator.

**Key Concepts**
* **Point Clouds**: A set of data points in 3D space that represent objects or scenes.
* **Iterative Closest Point (ICP)**: An algorithm used to align two point clouds by iteratively finding the closest points between them and refining the transformation.
* **Normal Distributions Transform (NDT)**: A probabilistic method for registering point clouds based on normal distributions of the data.
* **CARLA Simulator**: An open-source simulator for autonomous driving scenarios that allows users to create and work with point clouds.
