# Project Workflow

> Part of: **Scan Matching Localization**

## Additional Content

### Step 1. Log into VM Workspace

Open the VM workspace and log into the VM to practice the current project. 
Once you log into the VM, open a Terminal window. 

### Step 2. Clone the Project Github Repository
Fork the repository to your account and clone the

[Github repository](https://github.com/udacity/nd0013_cd2693_Exercise_Starter_Code)

to the workspace using the commands below.
  ```bash
  git clone https://github.com/udacity/nd0013_cd2693_Exercise_Starter_Code.git
  cd nd0013_cd2693_Exercise_Starter_Code/Lesson_7_Project_Scan_Matching_Localization/c3-project
  ```

### Step 3. Update the Project Code
Follow the instruction on the next page, **Project Steps**, to update the starter files. 


### Step 4. Execute the Project

When you finish updating the project files, you can execute the project using the instructions in the [project README](https://github.com/udacity/nd0013_cd2693_Exercise_Starter_Code/blob/main/Lesson_7_Project_Scan_Matching_Localization/c3-project/README.md).
