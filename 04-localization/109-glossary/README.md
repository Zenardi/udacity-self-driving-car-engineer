## Glossary

- **CARLA Simulator**: An [open-source simulator](https://carla.org/) useful for training and testing of various aspects of the autonomous vehicle technology stack, across computer vision, sensor fusion, localization, motion planning, etc. In the course, we utilized this to help create simulated 3D point cloud maps.
- **Point Cloud Library (PCL)**: A [powerful library](https://pointclouds.org/documentation/) for working with point clouds.

Additional term definitions around Iterative Closest Point (ICP) and Normal Distributions Transform (NDT) (and other related terms) can be found in the previous lesson's glossary on Creating Scan Matching Algorithms.
