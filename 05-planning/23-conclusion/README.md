## Summary

**Finite State Machines: A Solution to Behavior Planning**
===========================================================

This summary covers the key concepts and practical notes from a lesson on finite state machines.

### Key Concepts

* **Finite State Machine**: A mathematical model that can be used to solve behavior planning problems by representing a system's possible states and transitions between them.
* **State Space**: The set of all possible states that a system can be in. Finite state machines are well-suited for small state spaces, but may not perform as well with larger ones.
* **Behavior Planning Problem**: A problem where an autonomous agent must decide what action to take next based on its current state and the environment.

### Practical Notes

* When working with finite state machines, consider using them for problems with small state spaces, such as highway driving.
* For more complex scenarios like urban driving, other approaches may be more suitable.
* In the final project, you will have the opportunity to program a planner for highway driving and gain experience working with your own finite state machine.
