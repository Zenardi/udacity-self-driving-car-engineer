## Problem Statement
In this exercise you will practice 2 essential calculations necessary to build a
linear profile.

1. calculate the a final speed (v_f) given an initial speed (v_i), an
acceleration (a) and distance (d).

 v_f = sqrt(v_i ^ 2 + 2ad);


2. calculate the distance traveled (d) given an initial and final speeds (v_i & v_f)  and
acceleration (a).

 d = (v_f^2 - v_i^2)/ (2 * a);
