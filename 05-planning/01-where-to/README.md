## Summary

**Self-Driving Car Behavior Selection with Finite State Machines**
===========================================================

This project focuses on implementing a self-driving car's behavior selection using finite state machines. The goal is to determine the next action of the vehicle, such as changing lanes or taking an exit, rather than controlling individual steering wheel inputs.

### Key Concepts

* **Finite State Machine (FSM)**: A mathematical model that can be in one of a finite number of states, and can change from one state to another based on specific rules.
* **State Transition**: The process of moving from one state to another in an FSM.
* **Macroscopic Behavior**: High-level decisions made by the self-driving car, such as changing lanes or taking an exit.

### Practical Notes

To implement a finite state machine for behavior selection, you will need to:

* Define the states and transitions of the FSM
* Determine the rules for transitioning between states
* Implement the FSM in code using a programming language (e.g. Python)

Note: This project is not about controlling individual steering wheel inputs, but rather making high-level decisions about the vehicle's behavior.
