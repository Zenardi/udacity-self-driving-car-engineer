# Project Workflow

> Part of: **Motion Planning and Decision Making for Autonomous Vehicles**

## Additional Content

### Step 1. Log into VM Workspace

Open the VM workspace and log into the VM. 
Once you log in, open a Terminal window.

### Step 2. Clone the Project Github Repository
Clone the starter code

[Github repository](https://github.com/udacity/nd013-c5-planning-starter)

using the commands below:
  ```bash
  git clone https://github.com/udacity/nd013-c5-planning-starter.git
  cd nd013-c5-planning-starter
  ```
  As soon as you clone the repository, you must carefully read the [project README](https://github.com/udacity/nd013-c5-planning-starter/blob/master/README.md) file. This file has the instruction on how to execute the project.

### Step 3. Update the Project Code
Follow the instructions on the next page, **Project Requirements**, to update the starter files.

### Step 4. Execute the Project

When you finish updating the project files, execute the project and ensure that you fulfill all criteria and specifications mentioned in the [rubric](https://review.udacity.com/#!/rubrics/2949/view). When you submit your project, a human mentor will review your submission against the specific rubric criteria.
