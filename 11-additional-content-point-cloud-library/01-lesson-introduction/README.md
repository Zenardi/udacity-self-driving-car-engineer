## Lesson Introduction

In this lesson, we'll be covering a brief introduction to the [Point Cloud Library (PCL)](https://pointclouds.org/), an open-source library for working with 2D and 3D data, such as lidar point clouds.

This is an *optional* lesson - if you are familiar with PCL, you can feel free to skip it. The goal here is mostly just to get you comfortable with some basics of working with PCL, so that you can then move onto using PCL for localization in the later lessons.

This lesson was originally taught by Aaron Brown for the Sensor Fusion Nanodegree, and has been adapted for the current course.

In this lesson, you'll learn about:

- Point Cloud Data (PCD) files
- The Point Cloud Library (PCL)
- The PCL Viewer
- Creating and using a Lidar Object in PCL
- Templates and different point clouds
- Adjusting lidar parameters
- Examining the point cloud in PCL

On the next page, you'll catch up with Aaron as he moves into discussing PCD files, as we'll assume you already have some basics of lidar sensors and points in your toolkit.
