## Summary

**Course Prerequisites**
=========================

This README file summarizes the key concepts and prerequisites for a computer vision and machine learning course taught in Python. The course assumes proficiency in object-oriented programming, numpy, and matplotlib libraries.

**Key Concepts**
---------------

* **Object-Oriented Programming**: Understanding of classes, objects, inheritance, and polymorphism.
* **NumPy Library**: Familiarity with numerical computing and array operations.
* **Matplotlib Library**: Knowledge of data visualization tools for plotting graphs and charts.
* **Calculus**: Basic understanding of derivative calculations and their applications.
* **Linear Algebra**: Familiarity with matrix multiplication and dot products.

**Practical Notes**
------------------

To get the most out of this course, it's essential to have a solid grasp of these concepts. Python proficiency is also required as the entire course will be taught in Python. If you're new to object-oriented programming or numpy/matplotlib libraries, consider reviewing the basics before proceeding with the course material.

## Prerequisites
The prerequisites for this course are to be able to:

* Write object oriented code in **Python** and proficiently use the **numpy** and **matplotlib** libraries (such as for matrix multiplication and simple plotting)
* Calculate **derivatives of common functions** (basic calculus skills) such as

$x^2, \frac{1}{x}$

* Compute **multiplication of matrices and dot products** (basic linear algebra skills)
