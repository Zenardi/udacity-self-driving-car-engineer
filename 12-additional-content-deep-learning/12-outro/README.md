## Summary

**Deep Learning for Computer Vision: SDNs**
=============================================

This project focuses on building a deep neural network (SDN) using convolutional layers, skip connections, and up sampling through transposed convolutions.

### Key Concepts
* **Convolutional Layers**: These layers apply filters to the input data to extract features.
* **Skip Connections**: These allow the model to retain information from earlier layers and combine it with later layers' outputs.
* **Transposed Convolutions** (also known as **Upsampling**): These are used to upsample the feature maps, effectively increasing the spatial dimensions of the output.

No specific code or implementation details were provided in this lesson. However, the concepts introduced will be built upon in the next lesson, where fully convolutional networks are used for scene understanding and semantic segmentation tasks.
