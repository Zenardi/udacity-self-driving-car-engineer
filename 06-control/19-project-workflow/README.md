## Additional Content

## Control and Trajectory Tracking for Autonomous Vehicle

### Step 1. Log into VM Workspace

Open the VM workspace and log into the VM. 
Once you log in, open a Terminal window.

### Step 2. Clone the Project Github Repository
Clone the starter code

[Github repository](https://github.com/udacity/nd013-c6-control-starter)

using the commands below:
  ```bash
  git clone https://github.com/udacity/nd013-c6-control-starter.git
  cd nd013-c6-control-starter
  ```
  As soon as you clone the repository, you must carefully read the [project README](https://github.com/udacity/nd013-c6-control-starter/blob/master/README.md) file. This file has the details on the file structure and the instruction on how to execute the project.

### Step 3. Update the Code - Proportional-Integral-Derivative (PID)
Follow the instructions on the next page, **Project Requirements**, to update the starter files. 


Before you start coding, we strongly recommend you look at this

[rubric](https://learn.udacity.com/rubric/4957)

, against which the Mentor will review your submission. Your submission must satisfy all rubric criteria to pass the project; otherwise, the Mentor may ask you to re-submit. 
 


In this project, you will apply the skills you have acquired in this course to design a PID controller to perform vehicle trajectory tracking. Given a trajectory as an array of locations, and a simulation environment, you will design and code a PID controller and test its efficiency on the CARLA simulator used in the industry.

### Step 4. Execute the Project

When you finish updating the project files, execute the project and double-check that you fulfill all criteria and specifications mentioned in the [rubric](https://review.udacity.com/#!/rubrics/4957/view).
