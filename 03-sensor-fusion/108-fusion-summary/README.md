## Summary

**Kalman Filter Process for Tracking Moving Objects**
=====================================================

This README file summarizes the key concepts and practical steps involved in implementing a Kalman filter process for tracking moving objects, such as another car.

### Key Concepts
* **State Representation**: A 2D position and 2D velocity are used to represent the state of a moving object.
* **Prediction and Measurement Update**: The prediction step involves calculating the new state transition matrix F and process noise covariance Q based on the elapsed time ΔT. The measurement update step involves updating the state and covariance using the measurement function H and Jacobian.
* **Sensor Types**: The process distinguishes between camera and lidar sensors, with different measurement functions and Jacobians used for each type of sensor.
* **State Transition Matrix F**: A matrix that represents how the state evolves over time.
* **Process Noise Covariance Q**: A matrix that represents the uncertainty in the prediction step.

### Practical Notes
To implement this process, you will need to:

* Compute the elapsed time ΔT between measurements and use it to calculate the new state transition matrix F and process noise covariance Q.
* Use a nonlinear measurement function H for camera sensors or a constant lidar H matrix for lidar sensors.
* Update the state and covariance using the measurement function H and Jacobian.

Example code snippet:
```python
import numpy as np

# Define the state transition matrix F and process noise covariance Q
def calculate_F_and_Q(delta_t):
    # Calculate the new state transition matrix F
    F = np.array([[1, delta_t], [0, 1]])
    
    # Calculate the new process noise covariance Q
    Q = np.array([[delta_t**2, 0], [0, delta_t**2]])
    
    return F, Q

# Define the measurement function H for camera sensors
def nonlinear_measurement_function(x):
    # Compute the Jacobian of the measurement function
    J = np.array([[1, 0], [0, 1]])
    
    # Update the state and covariance using the measurement function H and Jacobian
    x_updated = x + J @ (measurement - H @ x)
    P_updated = J @ P @ J.T
    
    return x_updated, P_updated

# Define the constant lidar H matrix for lidar sensors
H_lidar = np.array([[1, 0], [0, 1]])

# Update the state and covariance using the measurement function H_lidar
x_updated = x + H_lidar @ (measurement - H_lidar @ x)
P_updated = H_lidar @ P @ H_lidar.T

return x_updated, P_updated
```

Here is our overall process inflow that combines what we've learned in this lesson. We are observing a moving object, for example, another car. Its state is represented by a 2D position and 2D velocity. Each time we receive new measurements from a sensor, we call the prediction and measurement update. Before we call the prediction, we have to compute the elapsed time between the previous observation and the new measurement.

Based on the elapsed time Delta T, we calculate the new state transition matrix F and the new process noise covariance Q. Then we predict stayed and covariance to the next timestamp. In the next step, we distinguish between camera and lidar. First, we have to transform the current data summation X from vehicle to censor coordinates as learned in the last video. If the current observation comes from a camera sensor, we have to use the nonlinear measurement function H for the state update, compute the new Jacobian H-J, and update the covariance with the current Jacobian.

Otherwise, if the current measurement is generated by a lidar sensor, we can simply use the constant lidar H matrix and update state and covariance as in the linear Kalman filter. We repeat this loop with the next measurement and so on.

## Fusion Summary
When a new measurement arrives, we perform the following steps:
- Calculate the time step $\Delta t$ and the new state transition matrix $\mathbf F$ and process noise covariance matrix $\mathbf Q$.
- Predict state and covariance to the next timestamp.
- Transform the state from vehicle to sensor coordinates.
- In case of a camera measurement, use the nonlinear measurement model and calculate the new Jacobian, otherwise use the linear measurement model to update state and covariance.
