## Summary

**Sensor Fusion System**
=========================

This project involves designing a fusion system that combines measurements from Lidar and camera sensors to perceive the environment and estimate position and velocity of other vehicles or pedestrians.

### Key Concepts
* **Kalman Filter**: A mathematical algorithm used for estimating the state of a system from noisy measurements.
	+ The Kalman filter was implemented in 2D as part of this project.
* **Nonlinear Measurement Models**: Mathematical models that describe how sensor measurements are related to the true state of the environment.
	+ Nonlinear measurement models were worked with as part of this project.
* **Sensor Fusion**: The process of combining data from multiple sensors to improve accuracy and robustness.

### Practical Notes
No specific code patterns or real-world applications were discussed in this lesson. However, it is expected that students will apply their knowledge of sensor fusion and Kalman filters to more complex scenarios, such as multi-target tracking, in the next lesson.
