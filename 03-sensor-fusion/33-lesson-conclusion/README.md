## Summary

**LiDAR Systems Summary**
=========================

This summary covers the key concepts and practical applications of LiDAR (Light Detection and Ranging) systems learned in the previous lesson. You should now have a solid understanding of the time-of-flight principle, which is the most widely used technology today.

### Key Concepts
* **Time-of-Flight Principle**: The method used to measure distance by calculating the time it takes for light to travel from the LiDAR sensor to an object and back.
* **FMCW (Frequency Modulated Continuous Wave) Technology**: A next-generation LiDAR technology that uses a modulated frequency signal to improve accuracy and range.
* **OPA (Optical Parametric Amplification) Systems**: Another emerging LiDAR technology that uses optical parametric amplification to enhance performance.
