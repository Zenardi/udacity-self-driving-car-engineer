## Summary

**Autonomous Vehicle Sensor Systems**
=====================================

This project explores the concept of autonomous vehicle sensor systems, specifically the use of multiple sensor types to achieve robust and reliable sensing capabilities. The goal is to create an autonomous system that can perform as well as human drivers under various conditions.

**Key Concepts**
---------------

* **Sensor fusion**: Combining data from multiple sensors with complementary properties to improve overall sensing capabilities.
* **Autonomous vehicle requirements**: Sensors must be able to operate effectively in diverse environments, including direct sunlight, heavy rain, night, snowstorms, and fully lit areas.
* **Primary sensor types**:
	+ Camera
	+ Radar
	+ Ultrasound
	+ LiDAR (Light Detection and Ranging)
* **LiDAR technology**: A relatively new sensor type with potential to become a game-changer in the autonomous vehicle industry.

**Practical Notes**
------------------

This project will involve understanding how each sensor type works, their differences, and where they can complement each other's sensing capabilities. The first lesson will cover camera, LiDAR, and radar basics, followed by a deeper dive into LiDAR technology and recent advancements in the field. The controversy surrounding the need for LiDAR sensors will also be explored, including Elon Musk's perspective on the matter.

Note: This project is not intended to provide code or implementation details but rather serve as an introduction to the key concepts and ideas presented in the lesson.
