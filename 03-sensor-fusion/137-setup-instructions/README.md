# Setup Instructions

> Part of: **Final Project: Sensor Fusion and Object Tracking**

## Additional Content

### Setup Instructions

The final project is based on the mid-term project, so you are already familiar with the setup. The [code files](https://github.com/udacity/nd013-c2-fusion-starter) are the same as for the mid-term - in fact, you can continue working on your code from before! The workspace in this lesson includes all the necessary files and requirements (although using the original starter code). So if you want to use the workspace, you can immediately jump to the next page!

If you want to run the project on your own computer, you probably have already downloaded the repository and installed the requirements in the mid-term project, so you should be ready to go! See the "Setup Instructions" page within the mid-term project for more details on setup instructions. Make sure that you did not make major changes in the mid-term project besides the given tasks. Alternatively, get a new clean checkout to avoid confusion.

### Using Pre-Computed Results
The final project uses pre-computed lidar detections in order for all students to have the same input data. **This is different data than used in the mid-term.** If you use the workspace, the data is prepared there already. Otherwise, [download the pre-computed lidar detections](https://drive.google.com/drive/folders/1IkqFGYTF6Fh_d8J3UjQOSNJ2V42UDZpO?usp=sharing) (~1 GB), unzip them and put them in the folder `results`.
