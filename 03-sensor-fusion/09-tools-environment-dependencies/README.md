## Tools, Environment & Dependencies
In most cases throughout the course, you will be provided Udacity workspaces which will have an environment installed for you. However, you may wish to also be able to perform exercises or complete the project in your own local environment. 

In that case, you will want to use the following tools and libraries:

*

[Python](https://www.python.org/downloads/)

*

[VSCode](https://code.visualstudio.com/download)

, or a similar IDE
* The Waymo Open Dataset (see previous page)
*

[TensorFlow](https://www.tensorflow.org/install)

*

[Open3D](http://www.open3d.org/)

*

[OpenCV](https://opencv.org/)

You can find further installation instructions in the course

[exercise repository](https://github.com/udacity/nd013-c2-fusion-exercises)

or

[project repository](https://github.com/udacity/nd013-c2-fusion-starter)

, including a `requirements.txt` file.
