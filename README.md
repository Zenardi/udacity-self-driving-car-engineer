## 📖 Course Overview

This [Nanodegree program](https://www.udacity.com/course/self-driving-car-engineer-nanodegree--nd0013) prepares you to design, build, and optimize self-driving car systems. You will learn the techniques that power autonomous vehicles across the full stack of capabilities, including computer vision, deep learning, sensor fusion, localization, path planning, and control. The program features hands-on projects using C++, Python, and deep learning frameworks, and was built in collaboration with industry-leading partners like Mercedes-Benz, Nvidia, and McLaren.

## ⏱️ Program Details

* **Skill Level:** Advanced
* **Estimated Time:** ~6 Months (at approx. 15 hours/week)
* **Pacing:** Self-paced learning

## 📋 Prerequisites

To succeed in this Nanodegree, incoming students should have a well-prepared background in:

* **Python (Intermediate):** Data structures, Numpy, Object-Oriented Programming (Classes).
* **C++ (Intermediate):** Memory Allocation, References, Classes, basic syntax.
* **Mathematics:** * Linear Algebra (Matrix operations/multiplication)
* Calculus (Derivatives, Integrals)
* Basic Statistics & Probability



## 🛠️ Skills Covered

* **Languages & Frameworks:** Python, C++, TensorFlow, OpenCV, Eigen Library
* **Core Concepts:** Computer Vision, Deep Learning, Convolutional Neural Networks (CNNs), Sensor Fusion (Radar & Lidar), Object Tracking, Path Planning, Robotics Control Systems
* **Tools & Algorithms:** Robot Operating System (ROS), Kalman & Particle Filters, PID Controllers, A* Search Algorithm, Finite State Machines

## 📚 Syllabus & Modules

### Course 1: Computer Vision

Learn to process raw digital images and apply basic-to-advanced computer vision techniques to find lane markings on the road.

* **Topics:** Camera calibration, image thresholding, color spaces, Hough transforms, Canny edge detection, and bird's-eye-view transformations.
* **Projects:** * *Detect Lane Lines*
* *Advanced Lane Detection*



### Course 2: Deep Learning

Leverage machine learning to solve autonomous vehicle perception problems. You will train models with real-world and simulator data.

* **Topics:** Neural Networks, TensorFlow, Convolutional Neural Networks (CNNs), Backpropagation, and Transfer Learning.
* **Projects:** * *Traffic Sign Classifier*
* *Behavioral Cloning* (Training a neural network to drive a vehicle in a simulator end-to-end).



### Course 3: Sensor Fusion

Learn how to fuse data from multiple sensors (Radar and Lidar) to track objects in the environment.

* **Topics:** Physics of sensors, Markov math, geometry/trigonometry for vehicular motion, and Kalman Filters.
* **Project:** *Extended Kalman Filters in C++* (Combine sensor measurements over time to estimate the position of other vehicles).

### Course 4: Localization

Implement foundational algorithms to ensure a vehicle knows exactly where it is in a complex environment.

* **Topics:** Markov Localization, Bayesian filters, vehicle motion models, and probabilistic sampling.
* **Project:** *Kidnapped Vehicle* (Implement a Particle Filter in C++ to localize a lost vehicle).

### Course 5: Path Planning

Search the environment, predict the behavior of other moving objects, and navigate the vehicle to its goal safely.

* **Topics:** Discrete path planning (A* search), prediction, behavior planning using finite state machines, and trajectory generation.
* **Project:** *Highway Driving* (Design an algorithm to navigate a car through highway traffic).

### Course 6: Control

Apply closed-loop controllers to actuate the vehicle and make it follow the generated trajectories.

* **Topics:** Actuators, steering, acceleration, and braking control.
* **Project:** *PID Controller* (Implement a Proportional-Integral-Derivative controller to safely maneuver the vehicle).

### Course 7: System Integration (Capstone)

Combine everything you've learned to run your code on a real or virtual self-driving car (Carla).

* **Topics:** Autonomous Vehicle Architecture, Introduction to the Robot Operating System (ROS), Packages, Catkin Workspaces, and writing ROS nodes.
* **Project:** *System Integration* (Work in a team environment to drive "Carla" around the Udacity test track).
